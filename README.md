
# Life Care

Life Care Hospital is one of the renowned and leading multispecialty hospitals comprising more than five topmost specialties



## Features

- Cross platform
- Responsive
- Authorization
- Authentication


## Tech Stack

**Framework:** Vite

**Client:** React, ChakraUI

**Server:** Node, Express

**Database:** MongoDB,Redis


## Run Locally

Clone the project

```bash
  git clone https://github.com/mujaheed13/nimble-sand-8219
```

Go to the project directory

```bash
  cd nimble-sand-8219
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`port`

`MongoUrl`

`GOOGLE_CLIENT_SECRET`

`GOOGLE_CLIENT_ID`

`key`

`sRound`

## API Reference

#### Welcome 

```http
  GET /home
```

#### Get | Post | Update | Delete User's

```http
  GET /user
```
#### Get | Post | Update | Delete Appointment's
```http
  GET /appointments
 ```
